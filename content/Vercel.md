# Vercel

[Vercel](https://vercel.com/) es la plataforma de desarrollo web y hosting de Next.js. Permite construir, desplegar y escalar aplicaciones web dinámicas.

## Qué es Vercel

- **Plataforma frontend**: specialize en React/Next.js
- **Deploy instantáneo**: Git push = deploy automático
- **全球 CDN**: 125+ puntos de presencia
- **Serverless functions**: Backend sin servidor

## Características principales

### Deployment
- Deploy desde Git (GitHub, GitLab, Bitbucket)
- Preview deployments por branch
- Rollbacks instantáneos
- Zero config para Next.js, React, Svelte, Vue, etc.

### Edge Functions
- Funciones serverless en edge
- Baja latencia global
- Soporte para AI streaming

### CDN y caching
- CDN global integrado
- Incremental Static Regeneration (ISR)
- Cache invalidation por tags

### Seguridad
- DDoS protection incluido
- L7 Firewall
- SSL automático

### AI Apps
- Optimizado para streaming de AI
- Integración con Vercel AI SDK
- Cache para respuestas de modelos

### Bases de datos (2026)

- **Vercel Postgres**: PostgreSQL serverless
- **Vercel KV**: Redis-compatible
- **Vercel Blob**: Almacenamiento de archivos
- **Marketplace**: AWS Aurora, DynamoDB

## Cuándo usar Vercel

- Apps Next.js/React
- Dashboards y web apps
- Sitios estáticos con dinámica
- AI apps con streaming
- Prototipos rápidos

## Precio

- **Free**: 100GB bandwidth, funciones limitadas
- **Pro**: $20/mes por usuario
- **Enterprise**: Custom
