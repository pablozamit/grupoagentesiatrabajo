# Apify

[Apify](https://apify.com/) es una plataforma en la nube para obtener datos de páginas web y automatización.

## Qué es

- **Web scraping**: Extraer datos de cualquier página web
- **Automatización**: Hacer tareas repetitivas automáticamente
- **20,000+ scrapers listos**: Herramientas ya hechas para common sites
- **Sin servidor propio**: Todo corre en la nube de Apify

## Para qué sirve

- **Investigación**: Recopilar datos de competidores
- **Leads**: Extraer contactos de directorios
- **Ecommerce**: Monitorizar precios de productos
- **Noticias**: Recopilar artículos de prensa
- **AI**: Alimentar modelos con datos de la web

## Cómo funciona

1. Eliges un Actor (scraper) del almacén o creas uno
2. Configuras qué datos quieres
3. Lo ejecutas en la nube
4. Descargas el resultado (CSV, JSON, etc.)

## Integraciones

- Google Sheets
- Zapier, Make
- Slack
- Webhooks
- APIs

## Enlace

- Web: [apify.com](https://apify.com/)

## 🚀 Cómo empezar
1. Crea una cuenta gratuita en [Apify](https://apify.com/).
2. Busca en la Store el actor **"Google Maps Scraper"** o **"Instagram Scraper"**.
3. Haz clic en **Try for free**, introduce el enlace de la web que quieres escanear y haz clic en **Start** para obtener los datos estructurados en Excel o JSON en segundos.
