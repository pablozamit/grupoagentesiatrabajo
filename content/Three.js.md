# Three.js

**Three.js** es una librería JavaScript para gráficos 3D en el navegador. Es la estándar más usada para crear experiencias 3D en la web.

## Qué es

- Librería de renderizado 3D basada en WebGL
- Abstrae la complejidad de WebGL
- Permite crear escenas, cámaras, luces, geometrías, materiales, animaciones
- Compatible con todos los navegadores modernos

## Usos típicos

- Juegos web
- Visualizaciones de datos 3D
- Experiencias interactivas
- Metaversos y mundos virtuales
- Configuradores de productos
- Sitios web inmersivos
- Arte digital
