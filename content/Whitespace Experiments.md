# Whitespace Experiments

[Whitespace Experiments](https://experiments.thisiswhitespace.com/) es una colección de experimentos de diseño web interactivo. El código es público y se puede copiar para aprender.

## Qué里面有

Experimentos visuales interactivos para inspirar y aprender:

- **Efectos 3D**: Three.js, WebGL
- **Partículas**: Sistemas de partículas en canvas
- **Shaders**: Efectos visuales personalizados
- **Animaciones**: Transiciones y efectos interactivos
- **UI**: Controles y elementos visuales innovadores

## Ejemplos destacados

- **Code Slice Hero**: Intro 3D con Three.js
- **Particle Phones**: Partículas con scroll
- **Cursor Nav**: Navegación 3D interactiva
- **Neural Burst**: Visualización de datos 3D
- **Mobius Strip**: Figura 3D experimental
- **Elastic Bars**: Física con canvas
- **Microchip Viewer**: Visor 3D interactivo

## Para qué sirve

- Inspiración para proyectos propios
- Aprender técnicas de animación
- Copiar código para tus proyectos
- Ver qué es posible en la web
