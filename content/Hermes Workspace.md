# Hermes Workspace

[Hermes Workspace](https://hermes-workspace.com) es una interfaz gráfica de usuario (PWA / Web App) diseñada específicamente para controlar de forma visual y centralizada al [[Hermes Agent]] (el backend y motor principal).

Actúa como un centro de mando completo que simplifica el desarrollo de proyectos y videojuegos mediante agentes de IA.

---

## Qué es (versión simple)

Mientras que el [[Hermes Agent]] funciona principalmente a través de la consola de comandos (terminal) o integraciones de mensajería (Slack, Discord, Telegram), el **Workspace** te proporciona un panel de control visual en tu navegador. 

Te permite ver en tiempo real qué está haciendo el agente, interactuar con él en una interfaz amigable de chat y gestionar todos tus archivos sin tocar una sola línea de consola.

---

## Características Principales

*   💬 **Chat Multi-modelo Avanzado**: Conversaciones fluidas con el agente y capacidad de cambiar de modelo con un clic.
*   🧠 **Visor de Memoria Persistente**: Permite inspeccionar qué recuerda el agente sobre tus preferencias, proyectos y clientes.
*   🛠️ **Gestor de Skills (Habilidades)**: Un editor visual para revisar, crear o modificar el catálogo de habilidades del agente.
*   💻 **Terminal y Explorador Integrados**: Ejecución de comandos del sistema y navegación del sistema de archivos en tiempo real desde la web.
*   📊 **Kanban y Misiones**: Un organizador de tareas integrado para coordinar los flujos de trabajo de tus proyectos.
*   🤖 **Swarms (Equipos de Agentes)**: Capacidad para desplegar múltiples agentes que trabajan de forma coordinada e independiente en un mismo espacio de trabajo.

---

## Comparativa: ¿Cuál usar?

| Característica | [[Hermes Agent]] Core | Hermes Workspace |
| :--- | :--- | :--- |
| **Tipo** | Backend / Motor de ejecución | Interfaz Gráfica (PWA) |
| **Interfaz** | Terminal o Apps de Mensajería | Panel de control web visual |
| **Facilidad** | Requiere mayor nivel técnico | Sencillo, intuitivo y visual |
| **Control de Archivos** | Manual (comandos de consola) | Explorador integrado con un clic |
| **Gestión de Equipos** | Manual por configuración | Sencilla e interactiva con Swarms |
| **Recomendado para** | Tareas ligeras / Automatización | Proyectos complejos y desarrollo de videojuegos |

---

## ¿Se pueden usar juntos?

**Sí, de hecho es la forma recomendada de trabajar.** El Hermes Workspace no sustituye al agente; se conecta directamente a él. El agente actúa como el cerebro que ejecuta los procesos de fondo en tu portátil o servidor (VPS), mientras que el Workspace te proporciona el control táctico y visual para guiarlo fácilmente.
