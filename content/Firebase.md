# Firebase

[Firebase](https://console.firebase.google.com/) es la plataforma de desarrollo de apps móviles y web de Google. Provee servicios backend sin necesidad de servidor propio.

## Servicios principales

### Firebase Realtime Database
- Base de datos en tiempo real (JSON)
- Sincronización instantánea a todos los clientes conectados
- Acceso directo desde móvil/navegador sin servidor intermedio
- Reglas de seguridad configurables

### Firebase Firestore
- Base de datos documental (NoSQL)
- Más moderna que Realtime Database
- Consultas potentes y escalable
- **Novedades 2026**: Firebase SQL Connect con soporte PostgreSQL, offline cache, y realtime sync

### Firebase Authentication
- Sistema de autenticación integrado
- Métodos soportados:
  - Email + contraseña
  - Teléfono
  - Google, Facebook, Twitter, Apple, GitHub
  - SAML / OpenID Connect (Enterprise)
- Multi-factor authentication (MFA) disponible

### Otros servicios

- **Firebase Hosting**: Hosting estático y CDN
- **Cloud Functions**: Funciones serverless
- **Cloud Storage**: Almacenamiento de archivos
- **Firebase ML**: Machine Learning en dispositivo
- **Firebase App Check**: Protección contra abuso

## Cuándo usar Firebase

- Apps móviles iOS/Android
- Web apps con backend necesario
- Prototipos rápidos
- Proyectos que necesitan realtime sync
- Startups que buscan velocidad

## Precio

- **Spark (Free)**: Incluye la mayoría de servicios
- **Blaze (Pay as you goes):** Escalable con Google Cloud

## 🚀 Cómo empezar
1. Entra en la [Consola de Firebase](https://console.firebase.google.com/) con tu cuenta de Google.
2. Haz clic en **Crear un proyecto** y sigue los sencillos pasos de configuración.
3. Activa el servicio **Firestore Database** o **Authentication** con un solo clic en la barra izquierda para tener tu base de datos y sistema de usuarios en la nube listos para usar.
