# Delphi Tools

Delphi Tools es una excelente colección de utilidades de diseño y desarrollo web que se ejecutan directamente en tu navegador de forma local.

### 🌟 Características clave
* **100% Gratuito:** Sin costes ocultos.
* **Sin registro (No Login):** No requiere crear cuenta de usuario.
* **Sin anuncios ni tracking:** Limpio y respetuoso con la privacidad.
* **Ejecución local:** Todo se procesa de forma segura en tu navegador.

### 🛠️ ¿Qué utilidades incluye?
Incluye una amplia gama de herramientas para desarrolladores y diseñadores:
* Editores y optimizadores de gráficos vectoriales (SVG).
* Herramientas de paletas de colores, degradados y contrastes.
* Utilidades tipográficas y formateadores de código.
* Generadores de sombras, bordes y layouts CSS.
